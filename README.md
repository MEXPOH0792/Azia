# Экспресс Азия Кафе — сайт

Сайт уйгурского кафе «Экспресс Азия» (Барнаул): главная с видео-hero, меню, онлайн-бронирование столика по схеме зала, контакты.

Статический сайт без сборки — чистый HTML/CSS/JS. Открывается локально двойным кликом и разворачивается на любом статическом хостинге.

## Структура

```
index.html        — основной сайт (использует images/ и videos/)
standalone.html    — автономная версия: фото и видео вшиты в один файл (≈3.5 МБ)
images/            — фото блюд и интерьера (.jpg — лёгкие, для сайта; .png — оригиналы; og.jpg — Open Graph 1200×630)
videos/
  hero.mp4         — фоновое видео hero (сжато, 720p, без звука)
  hero-source.mp4  — исходник видео (бэкап)
robots.txt
sitemap.xml
```

## Локальный запуск

Просто откройте `index.html` в браузере. Либо через локальный сервер:

```bash
python -m http.server 8000
# затем http://localhost:8000
```

## Что внутри (функционал)

- Переключение разделов (Главная / Меню / Бронирование / Контакты) без перезагрузки
- Мобильное бургер-меню, адаптив (брейкпоинт 760px)
- Кликабельный телефон (`tel:`)
- Бронирование: выбор столика на схеме зала, время, число гостей, форма, подтверждение
- Фоновое видео в hero (autoplay, muted, loop), уважает `prefers-reduced-motion`

## SEO / техбаза

`meta description`, Open Graph + Twitter Card, favicon, `theme-color`, schema.org `Restaurant` (JSON-LD), `robots.txt`, `sitemap.xml`.

## Деплой (GitHub Pages)

Settings → Pages → Build from branch → `main` / `root`. Сайт будет доступен по адресу вида `https://mexpoh0792.github.io/Azia/`.

> При деплое на реальный домен замените заглушку `https://express-asia.ru/` в `canonical`, `og:url`, `og:image` и `sitemap.xml` на ваш адрес (для Open Graph нужен абсолютный URL картинки).

## Статус

Форма бронирования — **демо без бэкенда**: показывает подтверждение локально, данные никуда не отправляются. Для приёма реальных броней нужен сервер / почта / Telegram-бот, а также реальная страница политики конфиденциальности (ссылка в форме ведёт на `#`).
